
💻 ServiceNow Laptop Request Management System

📌 Overview

This project automates the laptop request process using ServiceNow Flow Designer.

🚀 Features

- Service Catalog Item (Laptop Request)
- Manager Approval Flow
- Automatic Incident Creation
- Email Notification
- SLA Implementation

🔄 Workflow

1. User submits request
2. Manager approves
3. Incident created automatically
4. Notification sent to user
5. SLA starts tracking

🛠️ Tools

- ServiceNow
- Flow Designer
- Notifications
- SLA
