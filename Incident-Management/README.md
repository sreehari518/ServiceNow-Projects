🚨 Incident Management Project

---

📌 Description

This project demonstrates how incidents are created and managed in ServiceNow.

---

🎯 Use Case

Users can report issues which are tracked and resolved by the IT team.

---

⚙️ Features

- Incident creation
- Priority handling (High, Medium, Low)
- Assignment to users/groups
- State management (New → In Progress → Closed)

---

🛠️ Implementation Steps

1. Created incident records
2. Configured form fields
3. Implemented Client Scripts
4. Applied Business Rules

---

📸 Output

Screenshots are available in the screenshots folder.

---

💡 Learning Outcome

- Learned Incident lifecycle
- Understood ITSM workflow
- Hands-on experience with ServiceNow

---

🚀 Conclusion

This project helped me understand real-time incident management process in ServiceNow.
